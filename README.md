Readabo
=======

Browser extensions for simpler and more readable Weibo styles.

![](http://ww3.sinaimg.cn/large/61b0a91dgw1ebsrrw3yjej211d0jmwgx.jpg)

## Features
- Auto hide navigation bar
- Keyboard Shortcuts:
	- `.` Reload
	- `g` `h` Jump to Home
	- `g` `p` Jump to Profile
	- `g` `c` Jump to Comments
	- `g` `a` Jump to Mentions
	- `g` `m` Jump to DMs
